# Verdict — Elite Executive Decision-Intelligence Engine

Verdict is a high-performance, dark-mode AI web application engineered to eliminate cognitive overload and decision paralysis. Built with a FastAPI backend and a minimalist Vanilla JS frontend, Verdict streams analytical breakdowns in real time using the Google Gemini 2.5 Flash API.

---

## ✨ Features

- **Asynchronous Streaming**: Real-time response generation utilizing Gemini 2.5 Flash.
- **Structured Analytical Framework**: Standardized executive outputs including Quick Recommendations, Detailed Analysis, Pros & Cons, Risks & Trade-offs, and Execution Roadmaps.
- **Editorial Dark Theme**: Styled with modern typography (*Playfair Display* & *Inter*) and custom red bullet highlights.
- **Resilient Engine**: Built-in 20-second streaming timeout safeguards and graceful error handling.

---

## 🛠️ Tech Stack

- **Backend**: Python 3.10+, FastAPI, Uvicorn
- **AI Integration**: `google-genai` SDK (Gemini 2.5 Flash)
- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
